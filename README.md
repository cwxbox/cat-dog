# cat-dog

#### 介绍
一个开源的微信小程序，使用微信开发平台自带的内容管理进行内容的增删改查，更快速的方便大家的使用。这个程序主要用于对于流浪猫，流浪狗的信息登记和管理，让大家可以更好的了解关心身边的流浪动物。

#### 软件架构
软件架构说明

微信小程序原声开发+微信小程序中内容管理平台作为后台

#### 安装教程

将文件克隆到本地

```
git clone https://gitee.com/cwxbox/cat-dog.git
```

1、打开微信小程序导入项目

2、开通微信云服务

3、开通内容管理

4、在内容管理中使用内容管理
